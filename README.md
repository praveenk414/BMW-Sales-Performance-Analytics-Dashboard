# BMW M-Series Sales Performance Analytics Dashboard 

An interactive and dynamic Power BI dashboard designed to analyze and visualize the sales performance, pricing trends, and regional distribution of premium BMW M-Series and i-Series models.

## 🌟 Key Features
- **Dynamic Model Selection:** Features an interactive custom slicer to seamlessly switch between models (3 Series, 5 Series, i8, M5, etc.) with dynamic car image rendering.
- **Key Performance Indicators (KPIs):** Real-time tracking of Sum of Sales, Sum of Price (USD), Average Mileage, and Engine Size card metrics.
- **Temporal Analysis:** A dedicated line chart illustrating the "Sum of Price by Year" to track market and value trends over time.
- **Regional Breakdown:** Bar charts and pie charts showcasing sales volume and revenue across major global regions (Asia, North America, Europe, etc.).
- **Advanced Slicers:** Deep-dive filtering capabilities using Year, Fuel Type (Diesel, Electric, Hybrid, Petrol), and Vehicle Color.

## 🛠️ Tech Stack & Tools Used
- **Business Intelligence:** Power BI Desktop
- **Data Manipulation & Modeling:** Power Query, DAX (Data Analysis Expressions)
- **Design & UI:** Custom BMW M-Theme UI with integrated conditional image rendering.

---

## 📈 Key Insights Derived from the Dashboard
- **Model Highlights:** Provides granular tracking for high-performance models like the M5 and hybrid/electric icons like the i8.
- **Regional Dominance:** Helps identify which global market contributes the highest revenue and sales volume.
- **Fuel Type Pivot:** Easily tracks how sales trends are shifting from traditional Diesel/Petrol models toward Hybrid and Electric alternatives.
